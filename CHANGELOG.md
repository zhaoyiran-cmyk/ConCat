# ConCat 版本日志

## v1.2.5 (2026-04-04)

### 修复
- **左键拖窗误缩放**：无边框窗口在可调整大小时，边缘拖动会触发 `resize`，`fitScene()` 重算 `curScale` 看起来像「按住放大」。主进程将 **`resizable: false`**，尺寸仅由右键菜单倍率（`setBounds`）改变；并设 **`fullscreenable: false`** 减少误触全屏。
- **拖窗交互**：左键开始拖窗时对 **`mousedown` 调用 `preventDefault()`**，减轻默认行为带来的异常。

### 构建
- 输出目录 **release125**；`package.json` 版本 **1.2.5**

---

## v1.2.4 (2026-04-07)

### 键盘映射与桌面版可靠性
- **默认映射**：WASD 左摇杆；W 兼十字键上；QWER 十字（Q 左 / W 上 / E 右 / R 下）；JKL; 面键；UI L1/R1；TY L2/R2；ZX L3/R3；G 触摸板；H Home；Tab/Backspace Share/Option
- **备用键**：方向键与小键盘 8426 右摇杆；VBNM / FC / 12 / [] / 斜杠·引号·重音符 / P / 空格 等复用对应键位（多键同绑用集合去重）
- **全局钩子始终驱动映射**：不再用 `document.hasFocus()` 判断是否处理钩子事件，避免透明置顶窗误判导致「映射全丢」；与 Chromium `keydown` 并存时由 `Set` 去重
- **插着手柄时**：键盘状态与手柄输入 **合并** 到同一帧显示（按键 OR、摇杆键盘非零时覆盖、扳机取大）

### 锁定（点击穿透）
- 锁定后可通过 **整条顶部栏** 命中解锁；根据指针位置在 `setIgnoreMouseEvents` 与可点 UI 间切换（菜单 / F2 编辑区等）
- 主进程增加 `get-click-through`、`mouse-ignore-hit`；加载完成后同步锁定状态到界面

### 眨眼与眼部
- **闭眼**：去掉 `scaleY` 动画，眼睑图层 **瞬时显示约 120ms 后隐藏**
- **扳机**：**L2 / R2 按下沿** 分别只闭 **左 / 右** 眼；**不再**用 L1/R1 触发眨眼
- 键盘输入空闲时仍为 **双眼** 定时眨眼（与扳机无关）

### 其他
- **操作计数器**：小窗口缩放时数字更易读（字号与穿透模式下放大）
- 构建输出目录随版本：**release124**；`package.json` 版本 **1.2.4**

## v1.2.2 (2026-04-07)

### 眨眼与眼部显示
- **常态睁眼**：`eye_left_1` / `eye_right_1` 为闭眼/眼皮素材，猫头底图自带睁眼；常态 **隐藏眼皮图层**（`opacity: 0`），仅在眨眼关键帧中短暂显示，避免启动后一直像闭眼
- **眨眼触发**（v1.2.4 已改为扳机分眼 + 无 scale 动画；此处为 1.2.2 时行为）：**L1/R1 肩键按下沿**触发；键盘输入活跃时 **每 10 秒** 一次；眨眼动画时长 **0.26s**
- **F2 布局编辑器**：进入编辑时为眼皮图层增加 **`cat-eye-edit`**，便于对齐位置

### 布局与交互
- 默认 **左眼 / 右眼** 坐标与尺寸更新（`P.eyeL` / `P.eyeR`）
- **F2** 改为 **主进程全局快捷键** + IPC，无需窗口焦点即可开关布局编辑器；全局键钩中忽略 VK_F2，避免误触打字动画

## v1.2.0 (2026-04-03)

### 新功能
- **全局键盘监听**：使用 `uiohook-napi` 实现系统级键盘钩子，无论覆盖窗口是否有焦点，任何键盘输入都能触发猫猫打字动画
- **猫猫打字动画**：键盘按下时猫手上抬，松开后复位，左右手交替响应，支持多键同时按住
- **右键菜单**：右键弹出毛玻璃风格菜单（Stitch 设计），包含：
  - 置顶显示开关
  - 窗口大小切换（0.5x / 1.0x / 1.5x 单选）
- **拖动方式改为左键**：左键按住拖动窗口，右键改为呼出菜单

### 素材更新
- 猫身体替换为 `catheadwithouthand.png`（无手版本）
- 新增猫手素材 `cathand.png`，左手镜像显示
- 替换绿色按键素材 `btn_gree.png`
- 猫手层级调整到肩键上方（z-index:16）

### 移除
- 移除闭眼功能（eyeL/eyeR）
- 移除旧版肩键覆盖层（r1.png），由猫手替代

### 改进
- 布局编辑器移至窗口顶部居中，不再遮挡左侧内容
- 编辑器面板改为两列网格布局，增加详情栏（X/Y/W/H 输入框 + 加减按钮）
- 支持 Ctrl+方向键调整元素大小
- 修复右侧功能按键（fbT/fbR/fbB/fbL）无法在编辑器中拖动的 bug
